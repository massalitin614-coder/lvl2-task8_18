# Анаграммы

Пакет для группировки слов по признаку анаграммы на языке Go.

## Задача

На вход подаётся срез строк (слов на русском языке в Unicode). Необходимо:

- Привести все слова к нижнему регистру.
- Сгруппировать слова по множествам анаграмм (слова считаются анаграммами, если состоят из одного и того же набора букв).
- Исключить множества, содержащие только одно уникальное слово.
- Для каждого множества вернуть в качестве ключа первое встретившееся в исходном списке слово (в нижнем регистре), а в качестве значения — отсортированный по возрастанию срез всех слов множества (включая возможные дубликаты исходных слов).
