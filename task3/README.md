Данная программа реализует базовые возможности классической команды `sort` в Linux/Unix.  
Она сортирует строки текста, читая данные из файлов или стандартного ввода, и выводит результат.

## Функциональность

Поддерживаются следующие флаги (в точности как в GNU sort):

| Флаг | Описание |
|------|----------|
| `-k N` | сортировка по колонке (столбцу) номер N. Разделитель колонок – табуляция. |
| `-n`   | числовая сортировка (строки интерпретируются как числа). |
| `-r`   | обратный порядок сортировки (по убыванию). |
| `-u`   | выводить только уникальные строки (удаляет дубликаты). |
| `-M`   | сортировка по названию месяца (Jan, Feb, …, Dec). |
| `-b`   | игнорировать хвостовые пробелы при определении ключа сортировки. |
| `-c`   | не сортировать, а только проверить, отсортированы ли данные; если нет – вывести сообщение. |
| `-h`   | сортировка человеко‑читаемых размеров (K, M, G, T, P, E). |

Флаги можно комбинировать, например `-nr` означает числовую сортировку в обратном порядке.
