
Аналог классической Unix-команды `cut`, реализованный на Go.  
Программа читает данные из стандартного ввода, разбивает каждую строку по заданному разделителю и выводит только указанные поля (колонки).

## Возможности

- Поддержка флагов `-f`, `-d`, `-s` (аналогично оригинальному `cut`).
- Номера полей можно задавать списком (через запятую) и диапазонами (например, `1,3-5`).
- Если номер поля превышает количество полей в строке, оно игнорируется.
- Флаг `-s` позволяет выводить только строки, содержащие разделитель.
- Эффективная обработка больших файлов (потоковое чтение, буферизированный вывод).

## Установка и сборка

### Предварительные требования

- Установленный Go (версия 1.16 или выше).

### Сборка из исходного кода

1. Клонируйте репозиторий (или скопируйте файл `main.go` в пустую директорию).
2. Выполните в терминале:

```bash
go build -o cut main.go
В текущей папке появится исполняемый файл cut (на Windows — cut.exe).

Запуск без компиляции
Можно использовать go run напрямую:

bash
go run main.go [флаги]
Использование
text
cut -f "список_полей" [-d "разделитель"] [-s]
Флаги
Флаг	Описание	Значение по умолчанию
-f	Номера полей для вывода (обязательный). Можно указывать через запятую и диапазоны, например 1,3-5.	—
-d	Символ-разделитель полей.	Табуляция (\t)
-s	Выводить только строки, содержащие разделитель. Если флаг не указан, выводятся все строки (для строк без разделителя — вся строка как одно поле).	false
Примеры
Пример 1: базовое использование с разделителем :
bash
echo "apple:banana:cherry" | cut -d : -f 2
Вывод:

text
banana
Пример 2: несколько полей и диапазон
bash
echo "1,2,3,4,5" | cut -d , -f 1,3-5
Вывод:

text
1,3,4,5
Пример 3: разделитель по умолчанию (табуляция)
bash
printf "a\tb\tc\n" | cut -f 2
Вывод:

text
b
Пример 4: флаг -s — игнорировать строки без разделителя
bash
(echo "a:b"; echo "no delimiter") | cut -d : -f 1 -s
Вывод:

text
a
Вторая строка (no delimiter) не содержит :, поэтому она отброшена.

Пример 5: чтение из файла
bash
cut -d , -f 2-4 < data.csv
Пример 6: использование в конвейере
bash
ps aux | cut -f 1,11
