# Simple Unix Shell in Go

Этот проект представляет собой простой интерпретатор командной строки (shell), написанный на Go. Он поддерживает базовые возможности Unix shell: встроенные команды, запуск внешних программ, конвейеры (пайпы), перенаправление ввода/вывода, переменные окружения, условное выполнение (`&&`, `||`), а также обработку сигналов (Ctrl+C прерывает текущую команду, Ctrl+D завершает shell).

## Возможности

- **Встроенные команды:**  
  - `cd <path>` – сменить текущую директорию  
  - `pwd` – вывести текущую директорию  
  - `echo <args>` – вывести аргументы  
  - `kill <pid>` – отправить сигнал SIGTERM процессу  
  - `ps` – показать список процессов (вызов `ps aux`)

- **Внешние команды:** любые программы из `$PATH` (например, `ls`, `grep`, `cat`).

- **Конвейеры (пайпы):**  
  `command1 | command2 | command3`

- **Перенаправление ввода/вывода:**  
  `> file` – перезапись, `>> file` – добавление, `< file` – чтение из файла

- **Подстановка переменных окружения:**  
  `$VAR` или `${VAR}` заменяются на значения из окружения.

- **Условное выполнение:**  
  `command1 && command2` – выполнить command2, если command1 успешен  
  `command1 || command2` – выполнить command2, если command1 завершился с ошибкой

- **Обработка сигналов:**  
  Ctrl+C прерывает текущую выполняемую команду (не закрывает shell)  
  Ctrl+D (EOF) завершает работу shell

## Требования

- Go 1.16 или выше
- Unix-подобная операционная система (Linux, macOS) или Windows с установленным Git Bash / WSL (для корректной работы некоторых команд, например `ps`).  
  На Windows встроенные команды `ps` и `kill` работают только в среде, где доступны Unix-утилиты.

./myshell
Или запустить без компиляции:

bash
go run main.go
Примеры использования
bash
$ pwd
/home/user

$ cd /tmp
$ pwd
/tmp

$ echo Hello, $USER
Hello, user

$ ls -l > files.txt
$ cat files.txt

$ echo "line1" >> files.txt
$ cat files.txt

$ ps | grep go

$ sleep 10 && echo "finished"

$ false || echo "error occurred"

$ kill 1234   # отправить SIGTERM процессу 1234
